## 人流检测项目**演示**客户端

### 技术栈

1. react hooks + typescript
2. ant design UI library
3. create-react-app cli

### 本地测试

#### 安装依赖

如果没有代理请使用 `cnpm` 替代 `npm`。

```sh
npm install
```

#### 启动项目

```sh
npm start
```
